# La fabrique sociale

Site vitrine React + Vite pour une assistante sociale independante, construit autour de l identite visuelle `La fabrique sociale`.

## Lancer le projet

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
npm run preview
```

## Personnalisation rapide

Les informations a modifier en priorite se trouvent dans [src/App.jsx](/Users/gestionnairedeparcinformatique-valentin/Documents/Assitsant%20social/src/App.jsx):

- `site.phone`
- `site.email`
- `site.location`
- les textes des sections

## Identite visuelle

Le logo fourni a ete copie dans `public/brand/logo-board.jpg` pour etre reutilise dans la vitrine.
